# Sistema de Registro de Peticiones

## 📋 Descripción
Aplicación para registro y gestión de peticiones HTTP con:
- Backend Spring Boot
- Base de datos PostgreSQL
- Frontend Angular

## 🛠️ Tecnologías
- **Frontend**: Angular 19
- **Backend**: Spring Boot 3.x
- **Base de datos**: PostgreSQL 15+
- **ORM**: Hibernate/JPA

## 🚀 Instalación

### Prerrequisitos
- PostgreSQL 15+
- Java 17+
- Node.js 18+
